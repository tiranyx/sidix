# Roadmap

## Phase 0 — Repo & docs (sekarang)
- Dokumen visi/misi, PRD, ERD, arsitektur, governance.
- Skeleton repo + kontribusi.

## Phase 1 — MVP lokal (1–4 minggu)
- Chat (text) + model adapter (minimal 1 provider + 1 local option).
- Image generation (minimal 1 backend).
- STT/TTS minimal.
- Knowledge base RAG minimal + sitasi.
- Agent runner basic + log + permission gate.

## Phase 2 — Multi-user + safety (4–8 minggu)
- Auth multi-user + role.
- Rate limit + budget cap.
- Secrets encryption + redaction.
- Template agents.

## Phase 3 — Coding agent & tooling (8–12 minggu)
- Coding agent “patch-based” + test suggestions.
- Browser runner sandbox.
- Integrasi Git (opsional).

## Phase 4 — Ecosystem (12+ minggu)
- Plugin manifests untuk tools/adapters.
- Installer/presets (docker compose, one-click local).
- Observability lebih lengkap.
