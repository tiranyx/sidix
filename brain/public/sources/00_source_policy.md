# Kebijakan sumber (Source Policy)

Tujuan: semua riset yang kita masukkan ke `Mighan-brain-1` harus bisa dipertanggungjawabkan.

## Format sitasi (dua lapis: file + bibliography)
Kita pakai **dua-duanya**:
- **Sitasi ringkas di tiap file riset** (cukup untuk pembaca paham asal rujukan).
- **Daftar lengkap di** `brain/public/sources/bibliography.md` (single source of truth).

## Sumber yang diprioritaskan
- Dokumentasi resmi / blog engineering resmi (vendor/model)
- Paper (arXiv) dan model cards
- Release notes

## Aturan sitasi minimum
Setiap ringkasan riset wajib menyertakan:
- ID referensi (mis. `REF-2026-001`) yang cocok dengan `bibliography.md`
- Judul sumber
- Tanggal (kalau ada)
- Link
- Catatan singkat: “dipakai untuk apa”

## Aturan fakta vs opini
- Fakta: kutip/rujuk sumber
- Opini/hipotesis: label jelas sebagai opini/hipotesis

## Etika & lisensi
- Jangan copy-paste konten berlisensi tertutup ke repo publik.
- Kita boleh menulis **ringkasan** dan menautkan sumber.
- Kita tidak menyamarkan asal rujukan. Kita menghindari plagiarisme dengan:
  - menulis ringkasan dengan kata-kata sendiri,
  - menyertakan sitasi,
  - dan tidak menyalin paragraf panjang.
