## Data Structures
- Linked List, Stack, Queue, Hash Table, Array
- Tree: Binary Tree, BST, Full/Complete/Balanced/Unbalanced
- Graph: Directed, Undirected, Spanning Tree
- Representation: Adjacency Matrix, Adjacency List
- Heap (Min/Max)

## Asymptotic Notation
- Big-O, Big-Theta, Big-Omega
- Common Runtimes: Constant, Logarithmic, Linear, Polynomial, Exponential, Factorial

## Common Algorithms

### Recursion
- Tail Recursion vs Non-Tail Recursion

### Caches
- LRU Cache (Least Recently Used)
- MFU Cache (Most Frequently Used)

### Searching
- Binary Search, Linear Search

### Sorting
- Bubble, Selection, Insertion, Heap, Quick, Merge Sort

### Tree Algorithms
- Pre-Order, In-Order, Post-Order Traversal
- BFS, DFS

### Graph Algorithms
- BFS, DFS
- Bellman-Ford, Dijkstra's, A* Algorithm

### Backtracking
- Finding Hamiltonian Paths
- Solving N-Queen Problem
- Maze Solving
- The Knight's Tour Problem

### Greedy Algorithms
- Dijkstra's, Huffman Coding
- Kruskal's, Prim's Algorithms
- Ford-Fulkerson, Rabin-Karp, LFU Cache

### String Search & Manipulation
- Search Pattern in Text, Suffix Arrays
- Brute Force, Knuth-Morris-Pratt (KMP)
- Boyer-Moore, Robin-Karp, Substring Search

## Computer Architecture
- Floating Point Math
- Endianness: Big Endian, Little Endian
- Unicode, ASCII, Character Encodings

- Activity Diagrams, State Machine Diagrams
- Sequence Diagrams

- GoF Design Patterns (Creational, Structural, Behavioral)

- Dependency Injection, Null Object Pattern, Type Object Pattern

## Math Skills
- Probability, Combinatorics
- Complexity Classes: P, NP, NP-Hard, NP-Complete, P=NP
- Classic problems: Travelling Salesman, Knapsack, Longest Path

- Tries, Balanced Search Trees

- Horizontal vs Vertical Scaling
- Load Balancing, Clustering

## Architectural Styles

## Rekomendasi Urutan Belajar

1. **Dasar dulu**: Data structures (array, linked list, stack, queue, hash table)
2. **Algoritma dasar**: sorting, searching, recursion
3. **Complexity**: Big-O notation, analisis runtime
4. **Tree & Graph**: traversal, BFS/DFS, shortest path
5. **Design Patterns**: SOLID, GoF patterns
6. **System Design**: setelah cukup pengalaman praktis
7. **Advanced**: DP, backtracking, string algorithms
