# Skills (local packs)

Folder ini menyimpan **skill packs** (prompt/rules/recipes) yang bisa dipakai semua persona.

Konsep:
- Skill = *knowledge of how to do X* (bukan code execution).
- Bisa dipakai UI untuk menampilkan daftar skill/recipe, atau backend untuk routing.

Struktur bebas, contoh:

```
brain/skills/
  rag/
    SKILL.md
  hadith-validator/
    SKILL.md
```
